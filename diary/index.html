How it works

    어떤 채널이든 — 다시 보고 싶은 콘텐츠 발견

  2. 2

    URL 또는 캡쳐 + 짧은 메모를 봇에게 전송

    Claude가 채널·작성자·날짜·핵심 인용·맥락을 추출
